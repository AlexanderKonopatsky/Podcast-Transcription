# Папка Podcasts

Поместите ваши аудиофайлы подкастов в эту папку.

## Поддерживаемые форматы

- `.mp3`
- `.wav`
- `.ogg`
- `.m4a`
- `.flac`
- `.opus`
- `.webm`

## Использование

Для обработки всех файлов в этой папке:

```bash
python transcribe.py --batch --token YOUR_HF_TOKEN
```

Результаты будут сохранены в папке `output/`
