# Папка Output

Здесь сохраняются результаты транскрибации подкастов.

## Форматы вывода

Для каждого обработанного подкаста создаются 3 файла:

1. **`podcast_name.txt`** - читаемый текст с временными метками
   ```
   [00:15] Спикер:
     Добро пожаловать в наш подкаст!
   ```

2. **`podcast_name.json`** - полные данные в JSON формате
   - Сегменты с текстом и спикерами
   - Временные метки для каждого слова
   - Метаданные подкаста

3. **`podcast_name.srt`** - субтитры SRT формата для видеоплееров

## Использование

Результаты автоматически сохраняются в эту папку при запуске транскрибации.
