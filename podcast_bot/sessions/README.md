# Папка Sessions

Эта папка содержит приватные данные сессий пользователей и исключена из контроля версий по соображениям конфиденциальности.

## Содержимое

JSON файлы с историей диалогов пользователей с ботом. Каждый файл именован по ID пользователя Telegram.

## Структура сессии

```json
{
  "speaker": "Имя выбранного спикера",
  "history": [
    {
      "role": "user",
      "content": "Сообщение пользователя"
    },
    {
      "role": "assistant",
      "content": "Ответ бота в стиле спикера"
    }
  ]
}
```
