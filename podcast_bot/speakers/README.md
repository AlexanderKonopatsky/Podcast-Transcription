# Папка Speakers

Эта папка содержит приватные профили спикеров и исключена из контроля версий по соображениям конфиденциальности.

## Содержимое

JSON файлы с профилями спикеров, содержащие:
- Имя спикера
- Описание личности и стиля речи
- Примеры фраз и характерные высказывания

## Структура профиля

```json
{
  "name": "Имя спикера",
  "personality": "Описание характера и стиля",
  "speech_patterns": ["паттерн 1", "паттерн 2"],
  "style_examples": ["пример 1", "пример 2"]
}
```
