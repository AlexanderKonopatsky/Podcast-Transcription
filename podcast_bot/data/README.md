# Папка Data

Эта папка содержит приватные данные проекта и исключена из контроля версий по соображениям конфиденциальности.

## Содержимое

- `chunks.json` - кэш разбитых на сегменты подкастов с полным текстом
- `embeddings.npy` - эмбеддинги текстов для поиска

## Использование

Эти файлы создаются автоматически при первом запуске скрипта индексации подкастов:

```bash
python podcast_bot/scripts/index_podcasts.py
```
